# Grok Image Prompts — Tạo Ảnh cho Từng Slide

> **Hướng dẫn:** Copy từng prompt vào Grok (grok.com hoặc X/Grok) → chọn "Create image"
> 
> **Style chung:** Professional, clean, dark tech theme (dark blue/purple gradient), flat/isometric illustration, 16:9 ratio, no text on image.
> 
> **Lưu ý:** Thêm `--ar 16:9` hoặc chọn Landscape nếu Grok hỗ trợ chọn tỉ lệ.
